问题描述

(原题链接：LeetCode | 130. Surrounded Regions)

给定一个二维平面上的地图，全部由大写字母’X’和’O’组成。请将地图上由'O'组成且被'X'完全包围的区域全部设置为’X’。例如：

X X X X      X X X X
X O O X  =>  X X X X
X X O X      X X X X
X O X X      X O X X

解题思路

这是一个比较明显的搜索题目。最朴素的思想是你可以遍历地图上每一个点，如果为’O’的话就通过广度优先搜索找到与他连通的所有’O’，同时在搜索的时候判断是否有连通的’O’位于边界上，如果该连通块上没’O’位于边界上，那么将这个连通块上的’O’全部设置为’X’。

上面这种做法是完全可行的，不过代码略显复杂，你可以稍作变换，让你的解法变得更加优美简洁：

  1. 遍历边界，如果边界上存在’O’，则从这些’O’开始进行广度优先搜索，找到所有与它们连通的’O’，然后将它们全部标记为一个特殊的字符，例如’Y’，这样我们就找到了与边界连通的由’O’组成的区域；
  2. 遍历一遍地图，将剩余的’O’标记为’X’(因为这些’O’一定不与边界连通)，再将第(1)步中的’Y’全部设置回’O’。

以上方法的时间复杂度均为O(m*n)(m和n分别表示地图长宽)。通过这道题你可以发现，相同的解题思想稍微变换一些思路可以让你的代码质量变得更高。
